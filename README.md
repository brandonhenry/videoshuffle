# VideoShuffle

A simple Electron app for shuffling video files.

## Installation

```bash
npm install
```

**macOS Prerequisites**

If you don't have Homebrew installed, run:

```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

Then install FFmpeg:

```bash
brew install ffmpeg
```

## Usage

```bash
npm start
```
